# AGENTS.md

This directory contains project-wide documentation for ProjectSpire.

## Purpose

Use this directory for records, plans, and decisions that span multiple parts of the repository, such as workflows involving `Lab/`, `Apps/`, and `Mods/`.

Prefer this directory when a note is not specific to one subproject.

## Naming

Use numbered Markdown files:

```text
0001 - Title Words Here.md
```

Keep titles short and descriptive.

## Content

Each document should include:

- the date it was written or updated
- the project areas it concerns
- enough context to understand the decision later
- explicit open questions when a plan is not settled

Keep records concise. Link to narrower documentation in subdirectories when details already live there.
